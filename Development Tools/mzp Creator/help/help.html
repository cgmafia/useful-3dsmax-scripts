MaxScript Packager

Overview

MAXScript allows packages of related files, possibly making up a scripted tool, to be bundled and compressed into a single zip format file.

These package files can then be run like ordinary .ms files. Any secondary files, needed by the tool, will be either automatically found in the package or they will be automatically extracted to specified locations prior to running the main tool scripts.

The file type for script packages is '.mzp'.

Quick start

Creating a script package can be as simple as:

  1. Selecting source files and folders to be included in the zipping process
  2. Selecting which files should be run when the tool starts
  3. Export .mzp

However, you may also want to:

  1. Enter the particular details of your project
  2. Chnage the order that files will be loaded / run
  3. Add aditional commands to the Startup Script
  4. Change the way the final directies are stuctured
  5. Save the project for future updates
