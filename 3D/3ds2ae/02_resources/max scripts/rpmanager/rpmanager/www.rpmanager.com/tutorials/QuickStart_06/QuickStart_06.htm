Object Properties
 
 

Object Properties:

Object properties - assigned material, visible to camera, renderable, etc. - can be controlled by RPM per pass. Most commonly used are object material and light on/off, but many other properties - including properties of 3rd party objects like brazil lights - can be controlled on a pass by pass basis.

Some basic steps for turning on Material restoration per pass:

1. In RPManager, select a single pass - your base pass is probably a good one to start with although it doesn't matter.

2. Add a capture set in the Object Properties rollout.

3. If you didn't have the objects you want in the set selected and chose to have them auto added when you created the set, select the objects you want to add to the set, and use the 'add selected objects' to add them.

4. Press the 'define properties' button, and in the dialog that comes up select the material property in the 'object properties to capture' list. It should highlight and become bold. Ok this dialog.

5. Change to one of your other passes, and assign the materials you want to your objects for this pass.

6. Press the 'capture all objects in all sets properties' button (don't change passes or re-load this pass, your edits will be lost if you do). Ok the confirm, you should see a progress bar.

Change between the two passes to see the materials restored. It attempts to put the materials back into the medit as well, although if you do edit the materials you will need to re-capture (this will hopefully change in a future release of 3dsmax)

You can turn on the 'Auto Capture Properties' which removes the need to press the capture button when changing passes. Do not leave this on when not setting up properties as it will slow down RPM.
