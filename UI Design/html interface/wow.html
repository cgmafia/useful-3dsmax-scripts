This webbrowser control is associated with the WowController WOW control, which is set to be in "Controller Mode" and hidden.  WowController forwards all of the Events from IDocHostUIHandler and DWebBrowserEvents2 of the webbrowser through its events interface.  You use this mode of WOW when having problems tabbing around to other controls on the form.  The WOW control allows you to set your own "external" COM object so that you can call any method on your custom external object's interface directly from the webpage using Javascript.  The default custom interface that WOW provides has only two methods: RaiseEvent and RaiseEvent2.  Clicking on the HTML elements below cause these methods to be called, which in turn cause the WOW control's OnRaiseEvent and OnRaiseEvent2 events to be fired.
